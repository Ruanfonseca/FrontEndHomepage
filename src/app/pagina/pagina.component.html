TECNOLOGIAS QUE TRABALHO.

Front End

Em aplicações web no front end utilizo o AngularJS 16 com typescript com Arquitetura MVVM

Back End

Em aplicações web no Back end utilizo a linguagem Java em conjunto com o framework SpringBoot com Arquitetura Hexagonal.

Banco de dados

Banco de dados relacional PostgreSql

Broker

Sistemas de filas RabbitMQ

Infra Estrutura

Utilizo o Docker para criar os containers das aplicações e EC2's da AWS
